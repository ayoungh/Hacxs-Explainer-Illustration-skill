# Mascot Reference

## Character Definition

The default recurring figure in this skill is the bundled mascot from `references/assets/mascot.png`. Treat that image as the canonical identity for every generated illustration in this skill.

By default, the mascot appears in `cyberpunk-low-tech` scenes. The user may request:

- `avatar mode: default mascot`
- `avatar mode: minimal mascot`

## Visual Direction

- hooded silhouette
- dark faceplate with soft glowing eyes
- compact body and short limbs
- handheld laptop or hacker-tool posture when relevant
- calm, focused, slightly mysterious presence
- should stay recognizably close to the bundled reference image even when rendered in a sketchier style

## Behavioral Rule

If the mascot appears, it must perform the conceptual action:

- patching a cable
- routing a signal
- unlocking a gate
- carrying a packet
- inspecting a terminal
- feeding a machine
- guarding a threshold
- redirecting a path

If the image still works exactly the same after removing the mascot, the mascot is too decorative.

## Prohibitions

- no redesign that changes the mascot's silhouette or face
- no exaggerated emotional faces
- no cosplay-level costume detail
- no hero pose as the main point of the image
- no mascot that overwhelms the structure being explained
