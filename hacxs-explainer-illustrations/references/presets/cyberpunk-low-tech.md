# Preset: cyberpunk-low-tech

## Mood And Intent

Sparse, deadpan, low-tech hacker energy. It should feel like a hand-drawn mascot-assisted explainer from someone who spends a lot of time inside tools, terminals, pipelines, and weird systems, but still cares about clarity more than spectacle.

## Palette

- light base background
- black or charcoal line art
- restrained accent colors only
- use brighter accents for route emphasis, alerts, state changes, or signals
- never let color density dominate the image

## Line Quality

- slightly wobbly hand-drawn lines
- not polished vector art
- not rough to the point of chaos

## Object Vocabulary

- terminals
- cables
- ports
- patch bays
- route switches
- relays
- debug lamps
- locks
- vaults
- signal chutes
- odd maintenance rigs
- clipped command scraps

## Avatar Behavior

- default to `default mascot`
- keep the mascot visually consistent with `references/assets/mascot.png`
- keep the mascot small relative to the structure
- the mascot should route, patch, unlock, inspect, carry, or redirect something
- no hero-shot energy

## Composition Density

- default to `sparse`
- preserve a calm blank zone
- one dominant object family plus a small number of supporting elements

## Text Style

- short handwritten English labels only
- sharp, compact wording
- avoid slogans and long explanatory notes

## Anti-Patterns

- no glossy holo-UI boards
- no movie-poster neon glow
- no anime hacker character design
- no wall of monitors
- no black-background cyberpunk scene unless explicitly requested

## Prompt Fragments

- low-tech cyberpunk mascot explainer sketch
- clean white or very light background
- sparse hand-drawn terminal-and-cable metaphor
- restrained signal accents
- deadpan mascot doing real system work

## Preset-Specific QA

- If the scene would work better without any hacker vocabulary, the preset is being forced too hard
- If the scene looks like a stylish wallpaper instead of an explanation, simplify it
- If the glow or UI density increases, pull back toward cables, locks, relays, paper scraps, and repair logic
