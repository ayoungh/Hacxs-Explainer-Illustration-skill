# Preset: minimal-notebook

## Mood And Intent

Quiet, notebook-like, editorial, almost pre-product. This preset is for users who want the same explainer discipline with fewer hacker cues and a softer conceptual feel.

## Palette

- white background
- black or dark gray lines
- one restrained accent color if needed

## Line Quality

- loose but calm pen lines
- simple shapes
- no ornamental rendering

## Object Vocabulary

- sticky notes
- arrows
- drawers
- folders
- stamps
- ladders
- boxes
- paper scraps
- doors
- hooks

## Avatar Behavior

- default to `minimal mascot`
- mascot may be smaller and quieter than in `cyberpunk-low-tech`, but should still stay recognizable
- if present, the mascot should remain tiny and functional

## Composition Density

- sparse by default
- slightly softer and more open than the cyberpunk preset

## Text Style

- fewer labels than usual
- plain words, not slogan words

## Anti-Patterns

- no cyberpunk cues
- no dense infographic grids
- no polished brand-illustration treatment

## Prompt Fragments

- notebook-margin explainer sketch
- calm editorial hand-drawn layout
- sparse conceptual scene

## Preset-Specific QA

- If it feels too generic, add one stronger metaphor instead of more text
- If it looks like a productivity app illustration, strip it back and make it stranger
