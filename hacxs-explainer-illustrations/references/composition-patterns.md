# Composition Patterns And Originality Rules

## Core Structure Types

Choose one structure type per image.

### Workflow

Best for: input -> process -> output, AI pipelines, content production, system routing

How to draw it: a small number of steps with one dominant direction. Use the mascot only if the mascot helps the transition feel physical.

### System Close-Up

Best for: one local mechanism inside a larger system

How to draw it: show 3-5 modules only. Use ports, cables, relays, terminals, locks, or strange maintenance rigs instead of formal boxes whenever possible.

### Before/After Contrast

Best for: chaos/order, manual/automated, visible/invisible, blocked/running

How to draw it: left/right contrast or two compact scenes. Keep the visual change legible instantly.

### Character State

Best for: anxiety, overload, doubt, confidence, failure-state to stable-state shifts

How to draw it: 2-4 compact states with one short label each. Avoid making it look like a meme panel.

### Concept Metaphor

Best for: abstract judgments, trust, filtering, compounding, validation, signal extraction

How to draw it: one strange-but-valid object or machine with a small amount of input and a clear result.

### Layered Method

Best for: frameworks, capability stacks, layered reasoning, system maturity

How to draw it: stack modules or surfaces without turning them into a formal pyramid slide.

### Route Map

Best for: handoffs, progress paths, launch routes, learning paths

How to draw it: a line, path, tunnel, cable route, or corridor with a few stops only.

### Mini Comic Sequence

Best for: process commentary, cautionary contrast, failure to success sequences

How to draw it: 2-4 tiny scenes, one action per scene.

## How To Invent A Fresh Metaphor

1. Turn the abstract idea into a physical action: leaking, routing, locking, throttling, splitting, patching, buffering, tracing, validating, passing through.
2. Turn the system into a tactile object: terminal, relay box, patch bay, vault, socket wall, cable bundle, hatch, chute, signal gate, odd workstation.
3. If the mascot is present, make the mascot do the work instead of observing it.

## Useful Object Pool

- terminal
- relay box
- signal cable
- vault door
- patch bay
- socket wall
- debug lamp
- port array
- chute
- lock
- route switch
- control strip
- notebook margin
- maintenance rig

Choose only 1-2 major object families per image.

## Anti-Repetition Rules

- do not reuse the same terminal-and-cables scene for every article
- do not turn every workflow into a glowing screen
- do not default to the same left-middle-right composition if another metaphor would be fresher
- do not let cyberpunk cues become decoration detached from the concept
- do not copy compositions from other example sets unless the user explicitly asks for recreation or editing
